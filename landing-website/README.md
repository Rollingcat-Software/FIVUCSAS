# FIVUCSAS Landing Website

Official landing page for the FIVUCSAS biometric identity verification platform.

**Live**: [fivucsas.com](https://fivucsas.com)

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS 3
- Framer Motion (animations)
- Vite 6

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Output is written to `dist/`. Deploy this folder to your web server.
