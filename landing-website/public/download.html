Install instructions

Linux (.deb)

On Debian, Ubuntu, Linux Mint, Pop!_OS, and other Debian-family distros:

sudo apt install ./fivucsas_1.0.0_amd64.deb

Using apt install ./file.deb (with the leading ./) lets apt resolve any missing runtime dependencies automatically. On pure dpkg systems: sudo dpkg -i fivucsas_1.0.0_amd64.deb && sudo apt -f install.

Windows (.msi)

Heads up: the Windows installer is currently unsigned. Microsoft Defender SmartScreen will show a blue "Windows protected your PC" banner the first time you run it. Click More info → Run anyway to continue. An Authenticode signature will be added once our code-signing cert is provisioned.

Double-click FIVUCSAS-1.0.0.msi, follow the installer wizard, pick an install directory if you want, and FIVUCSAS will appear under the Start menu group FIVUCSAS with a desktop shortcut.

Android (APK)

We distribute the APK via GitHub Releases until we publish to Google Play. On your phone: download the APK, open it, and when prompted enable Install unknown apps for your browser (Settings → Apps → your browser → Install unknown apps). Android will then let you tap through the install.

Verify your download

Every release is checksummed. The Linux .deb is additionally signed with our GPG release key 3ED5 3CCD F292 AF22 E71F AB85 CCDB BE54 7D74 CA7A.

SHA256 (all platforms)

# Compare the line below with the sha256 shown on this page / in GitHub Releases.
sha256sum fivucsas_1.0.0_amd64.deb

GPG signature (Linux .deb)

# One-time: import our release key
curl -sL https://fivucsas.com/pgp.asc | gpg --import

# Verify the detached signature packaged alongside the .deb
gpg --verify fivucsas_1.0.0_amd64.deb.sig fivucsas_1.0.0_amd64.deb

# Or, if you have dpkg-sig installed, verify the in-archive signature:
dpkg-sig --verify fivucsas_1.0.0_amd64.deb

A valid signature prints Good signature from "FIVUCSAS Release <release@fivucsas.com>". Anything else — treat the file as compromised and re-download.

Build from source

All clients are open source. The desktop app is Kotlin Multiplatform + Compose Desktop, the Android app is KMP + Jetpack Compose. 

git clone https://github.com/Rollingcat-Software/FIVUCSAS.git
cd FIVUCSAS/client-apps
./gradlew :desktopApp:packageDeb     # Linux installer
./gradlew :desktopApp:packageMsi     # Windows installer (needs Windows + WiX)
./gradlew :androidApp:assembleRelease
