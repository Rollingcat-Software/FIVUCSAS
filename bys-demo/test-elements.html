FIVUCSAS Web Components Phase C

Vanilla test page for <fivucsas-verify> and <fivucsas-button> custom elements. No framework required — works with any HTML page.

<fivucsas-button> — Modal sign-in button

Opens a modal containing the verify widget. Closes automatically on success or cancel.

<fivucsas-button
  tenant-id="marmara"
  client-id="test-client"
  redirect-uri="https://demo.fivucsas.com/callback"
  label="FIVUCSAS ile Giriş Yap"
  theme="light"
  lang="tr">
</fivucsas-button>

<fivucsas-verify> — Inline embed

Place the verify widget directly in your page without a button/modal.

<fivucsas-verify
  tenant-id="marmara"
  client-id="test-client"
  redirect-uri="https://demo.fivucsas.com/callback"
  theme="light"
  lang="tr"
  width="380px"
  height="560px">
</fivucsas-verify>
